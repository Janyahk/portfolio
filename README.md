# 🌟 My Portfolio Website

This is my personal portfolio website built using **React + Vite**.  
It showcases my skills, projects, experience, and contact information in a clean and interactive UI.

---

## 🚀 Tech Stack

- **React.js**
- **Vite**
- **Tailwind CSS**
- **JavaScript (ES6+)**
- **GitHub Pages (Deployment)**

---

## 📸 Features

- 💼 About Me & Skills Sections  
- 🖼️ Projects Showcase  
- 📞 Contact Form   
- 📱 Fully Responsive Design  
- ⚡ Optimized build with Vite
